How to run this example
=======================
1) git clone https://github.com/ShambuGIT/spring-boot-gradle-examples.git OR download source file 
2) Import spring_rabbitmq_batch_message as gradle
3) Setup a rabbitmq in localhost OR change the rabbitme details in application.yml
4) Compile and run the program.
You will see my.batch.queue created in rabbitmq and this program will start sending the message to queue and consume as a batch.
